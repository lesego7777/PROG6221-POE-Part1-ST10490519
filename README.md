## How to Run
1. Clone this repository.
2. Open the solution file (`.sln`) in Visual Studio.
3. Ensure `greeting.wav` is in the output directory (set "Copy if newer").
4. Build and run (F5).

## Features
- Voice greeting on startup (WAV file).
- ASCII art logo display.
- Personalised interaction (asks user's name).
- Responses on password safety, phishing, safe browsing, and privacy.
- Input validation for empty or unrecognised queries.
- Enhanced console UI with colours and formatting.
